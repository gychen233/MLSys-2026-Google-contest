# MLSys-2026-Google-contest

The thoughts and implementation of the MLSys 2026 Google contest. (Track A Rank 5)

The .zip file in this repository is my official submission.

There is an executable file, source code, and a PDF about my ideas in the zip, but I forgot to include json.hpp.

The biggest advantage of my implementation is its speed. I specifically use DP, decision monotonicity, etc. to optimize it.

However, partly due to a misunderstanding of the rules, I exceeded some boundaries and received a poor score in the official test. I think my implementation could achieve a better score after some adjustments.

The other code consists of attempts during the competition.
